Help

Welkom bij Holy Ghostwriter. Hieronder vind je een korte handleiding.

1. Starten

  • Open de homepage en maak een nieuw preekproject.
  • Vul je bijbelreferentie en inhoudsbeschrijving in.
  • Gebruik de knoppen om ideeën of een volledige preek te genereren.

2. Instellingen

  • Stel je OpenRouter API key in.
  • Kies en beheer AI modellen.
  • Pas prompts aan voor stijl en voorkeuren.

3. Veilig gebruik

Deel geen persoonsgegevens of vertrouwelijke pastorale informatie in AI prompts. Gebruik altijd je eigen oordeel en controleer inhoud theologisch en pastoraal.

4. Problemen tegengekomen?

Loop je vast of heb je een bug gevonden? Meld het dan via GitHub issues of neem contact op via e-mail.

  • Meld een issue op GitHub
  • Stuur een e-mail naar support

5. Veelgestelde vragen (FAQ)

Waarom zie ik een fout over API key of autorisatie?

Controleer in Instellingen of je OpenRouter API key correct is ingevuld en actief is.

Waarom krijg ik meldingen over credits of limieten?

Je OpenRouter-account heeft mogelijk onvoldoende credits of een rate limit bereikt. Voeg credits toe of probeer later opnieuw.

Hoe kies ik het juiste model?

Vergelijk kosten en ondersteunde bestandstypes in Instellingen. Kies een model dat past bij je budget en of je PDF/afbeeldingen wilt gebruiken.

Welke contextbestanden kan ik uploaden?

Tekstbestanden zijn altijd geschikt. PDF en afbeeldingen werken alleen bij modellen die deze typen ondersteunen.

Wordt mijn onboarding opnieuw gestart?

Onboarding start wanneer je de homepage of preekpagina opent met de parameter ?onboarding in de URL.

Hoe betrouwbaar is de gegenereerde inhoud?

Gebruik AI-output als hulpmiddel. Controleer altijd inhoud, bijbeluitleg en toon voordat je publiceert of preekt.
